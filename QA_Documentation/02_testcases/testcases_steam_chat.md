# Тест-кейсы для чата с привязкой Steam

## TC-01: Успешная регистрация
- **ID:** TC-01
- **Название:** Регистрация нового пользователя
- **Предусловия:** Сервер запущен, страница `/register` открыта
- **Шаги:** 
  1. Ввести логин `Admin123`
  2. Ввести пароль `Qwerty123`
  3. Нажать "Зарегистрироваться"
- **Ожидаемый результат:** Flash "Регистрация успешна! Теперь войдите." Редирект на страницу входа
- **Фактический результат:** Соответствует
- **Статус:** Pass

## TC-02: Вход с корректными данными
- **ID:** TC-02
- **Название:** Вход существующего пользователя
- **Предусловия:** Пользователь `Admin123` создан
- **Шаги:** 
  1. На `/` ввести логин `Admin123`
  2. Ввести пароль `Qwerty123`
  3. Нажать "Войти"
- **Ожидаемый результат:** Перенаправление на `/dashboard`, приветствие "Добро пожаловать, Admin123!"
- **Фактический результат:** Соответствует
- **Статус:** Pass

## TC-03: Привязка Steam через числовую ссылку
- **ID:** TC-03
- **Название:** Привязка публичного Steam профиля
- **Предусловия:** Пользователь авторизован, Steam не привязан
- **Шаги:** 
  1. На `/dashboard` вставить `https://steamcommunity.com/profiles/76561199444289707/`
  2. Нажать "Привязать Steam"
- **Ожидаемый результат:** Flash "Steam профиль привязан!" Отображаются аватар и имя
- **Фактический результат:** Соответствует
- **Статус:** Pass

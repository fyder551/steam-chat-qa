# Баг-репорт: Вставка изображения в чат не отображается

**ID:** BUG-001  
**Заголовок:** При вставке скриншота (Ctrl+V) в поле ввода чата отображается только текст ссылки

**Окружение:** Windows 11, Chrome 124, сервер localhost:5000

**Шаги для воспроизведения:**
1. Авторизоваться, Steam привязан.
2. Открыть чат с другом.
3. Сделать скриншот (Ctrl+Shift+S) и скопировать в буфер.
4. В поле ввода нажать Ctrl+V.
5. Нажать "Отправить".

**Фактический результат:** Отправляется текст `/static/uploads/...png` вместо картинки.

**Ожидаемый результат:** Должна отображаться картинка, при клике открываться в новой вкладке.

**Серьёзность:** Major  
**Приоритет:** High

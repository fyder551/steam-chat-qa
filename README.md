# Steam Chat QA Portfolio

Этот репозиторий содержит проект веб-чата с интеграцией Steam и артефакты тестирования (чек-листы, тест-кейсы, баг-репорты). Разработан в рамках обучения QA.

## Структура
- `QA_Documentation/` – чек-листы, тест-кейсы, баг-репорты
- `site steam/` – исходный код веб-приложения (Flask, WebSocket)

## Основной функционал
- Регистрация и авторизация (валидация логина)
- Привязка профиля Steam (через ссылку /profiles или /id)
- Отправка/принятие заявок в друзья
- Создание открытых/закрытых групп, управление группами
- Личный и групповой чат с WebSocket
- Статусы онлайн/офлайн
- Загрузка и отправка изображений (Ctrl+V)

## Как запустить проект
1. Установить Python 3.8+
2. Установить зависимости: `pip install -r requirements.txt` (или вручную: flask, flask-socketio, eventlet, requests, werkzeug)
3. Запустить `python app.py` в папке `site steam`
4. Открыть `http://127.0.0.1:5000`

## Выявленные баги (примеры)
- Вставка изображения через Ctrl+V не отображается (только ссылка)
- Задержка доставки сообщений в чате
- Клик по аватарке друга не ведёт в Steam профиль

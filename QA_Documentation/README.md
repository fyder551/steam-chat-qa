# QA Portfolio – тестирование чата с интеграцией Steam

Этот репозиторий содержит артефакты тестирования веб-приложения (чат с авторизацией, привязкой Steam, группами, WebSocket).

## Содержимое
- `checklists/` – чек-листы (регрессионный)
- `testcases/` – тест-кейсы (регистрация, вход, привязка Steam)
- `bugreports/` – баг-репорты (вставка изображений)

## Выявленные дефекты
1. Вставка изображения через Ctrl+V не работает (отображается ссылка)
2. Задержка доставки сообщений в чате (5-10 сек)
3. Клик по аватарке друга не ведёт в Steam профиль
